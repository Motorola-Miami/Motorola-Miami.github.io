    Extras Magisk, KernelSU...¶

    You can safely flash the latest Magisk release without any problems directly from TWRP or LineageOSRecovery.

    WARNING: If you care about banking apps / others that contains root checks, don't flash Magisk since it will break device integrity. That can cause unwanted results such as missing apps on Google Play Store

    You can get root without problems, just install the latest KernelSU release in your device.
