    Flashing¶

    1) Booting into TWRP Recovery)¶

    • Download the TWRP.img from HEREto your PC and ROM file to your phone or a pendrive.
    • Power off the device, and boot it into bootloader mode:
    • With the device powered off, hold Volume Down + Power. Keep holding both buttons until the word BOOTLOADER appears on the screen, then release.
    • Boot into the downloaded TWRP.img to your device by typing in terminal/cmd:
      •     fastboot boot TWRP.img
    • This will take you to TWRP recovery. Flash the ROM from your storage or using a pendrive.
    • (Optionally): If you want to install any add-ons like Gappsflash it like you did for the ROM from TWRP.
    • Once you have installed everything successfully, click Reboot then click system.
    • If you want to update from the previous version of the same ROM to the new version (dirty flash), follow the same steps except for don't format data. #Note: If in a release changelog it is mentioned that you can't dirty flash then don't or you'll end up bootlooping.

    2) Flashing Using Provided ROM's or lineageOS Recovery)¶

    • Download the lineageOS or provided ROM's Recovery (from the Download Page) to your PC and ROM file to your computer.
    • Make sure you have the following files:
      •     boot.img
    dtbo.img
    vendor_boot.img
    ROM-xxxxxxxx-UNOFFICIAL-miami.zip
    • Power off the device, and boot it into bootloader mode:
    • With the device powered off, hold Volume Down + Power. Keep holding both buttons until the word BOOTLOADER appears on the screen, then release.
    • Flash the downloaded recovery files to your device by typing in terminal/cmd:
      •     fastboot flash boot.img
    fastboot flash dtbo.img
    fastboot flash vendor_boot.img
    • Now navigate to recovery using volume and power buttons. After Entering recovery do Factory Data Reset then click Apply Update and then Adb Sideload then connect your phone to PC.
      •     adb sideload ROM-xxxxxxxx-UNOFFICIAL-miami.zip
    • (Optionally): If you want to install any add-ons like Gappsflash it like you did for the ROM from terminal/cmd.
    • Once you have installed everything successfully, click Reboot System Now.
    • If you want to update from the previous version of the same ROM to the new version (dirty flash), follow the same steps except for don't format data. #Note: If in a release changelog it is mentioned that you can't dirty flash then don't or you'll end up bootlooping.

    About Device¶

    The Motorola Edge 30 Neo (codenamed "miami") is a mid-range smartphone from Motorola. It was released in October 2022.
